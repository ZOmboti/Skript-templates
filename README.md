Skript-templates is a project started on 2/21/2026 managed solely by me. 
As of now I've uploaded 4 templates that help you get started with you skripting journey.

Templates:

command template:
  Has guide for everything you can tweak in a command in skript, can be replaced with your code

function template:
  Explains how to make and use a function, can be replaced with your code.

loops template:
  Explains how to use loop players and loop "x" times loops, can be replaced with your code.

variables template:
  Explains how to use variables, can be replaced with your code.


If I managed to help you I'm already happy, feel free to use and share these templates anywhere.
